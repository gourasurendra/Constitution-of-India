# 🇮🇳 Indian Constitution

An interactive web-based platform designed to provide easy access to the Constitution of India, including all Articles, Parts, Schedules, Amendments, Fundamental Rights, Duties, and Directive Principles.

## 📖 About the Project

This project aims to make the Constitution of India accessible, searchable, and easy to understand for students, educators, UPSC aspirants, legal professionals, and citizens.

The website provides a modern and interactive interface for exploring constitutional provisions and learning about India's legal framework.

## ✨ Features

* 📜 Complete Constitution of India
* 🔍 Powerful Search Functionality
* 📚 Browse Articles, Parts, and Schedules
* 🌙 Dark Mode Support
* 📱 Fully Responsive Design
* ⚡ Fast and User-Friendly Interface
* 🎯 Ideal for UPSC, SSC, APPSC, TSPSC, and Competitive Exam Preparation
* 🔖 Article-wise Navigation
* 📖 Educational and Reference Resource

## 🏛️ Constitutional Content

The platform includes:

* Preamble
* Fundamental Rights
* Directive Principles of State Policy
* Fundamental Duties
* Union Government
* State Government
* Judiciary
* Emergency Provisions
* Constitutional Amendments
* Schedules
* Important Articles

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/gourasurendra/indian-constution.git
```

### Open the Project

```bash
cd indian-constution
```

Open `index.html` in your browser.
